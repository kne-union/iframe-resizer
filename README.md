
# iframe-resizer


### 描述

用于自动计算内部iframe的高度


### 安装

```shell
npm i --save @kne/iframe-resizer
```

### 示例


#### 示例样式

```scss
.ant-card {
  border-color: black;
  text-align: center;
  width: 200px;
}
```
